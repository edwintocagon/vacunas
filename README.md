# Inventario de vacunación de empleados

## 1.\_ reconstruir proyecto

`yarn`

## 2.\_ levantar servidor Fake API (json-server)

`yarn run json-server --watch src/api/db.json --port 5000`

o tambien ` yarn fake-api`

## 3.\_ ejecutar proyecto

`yarn run dev`
